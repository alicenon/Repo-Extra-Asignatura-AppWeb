Como se puede observar en estos ejemplos, con la etiqueta p se pueden definir párrafos. Cada definición de párrafo provoca un salto de línea más grande de lo normal. Además existe el atributo align, el cual nos permite establecer la alineación que deseemos al párrafo. Se puede observar que la alineación justify, que es la de este párrafo, funciona. Para comprobarlo observar el párrafo de abajo que es idéntico a éste, sin embargo no se le ha aplicado esta justificación.
